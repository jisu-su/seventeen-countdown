# WOOZI Countdown (Lite)

> **세븐틴 우지(이지훈)의 전역을 기다리는 캐럿들을 위한 약식 카운트다운 페이지**

이 브랜치는 배포를 위한 약식 버전입니다. 식단표, 음악, 소통창 기능을 제외하고 오직 카운트다운 화면만 제공합니다.

---

## ✨ 주요 특징

- **심플한 구성**: 오직 카운트다운 타이머만 노출됩니다.
- **최적화**: 불필요한 스크립트와 에셋을 제거하여 로딩 속도를 높였습니다.
- **디자인**: 기존 세븐틴 테마(로즈쿼츠 & 세레니티) 그라데이션 배경을 유지합니다.

---

## 🛠 기술 스택

- HTML5
- CSS3 (Flexbox, CSS Variables)
- Vanilla JavaScript
